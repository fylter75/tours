Tour Travels

This module provides the functionality to plan tour for a group

Manage your contract with suppliers for Accommodation,Transportation,Restaurants,Guides.

Configure accommodation prices based on seasons.

Manage and Display your package with Itinerary and other details on website page.

Calculate Group costing, costing for single person, per person in double, per person in TPL and person with child.

Calculate Selling price by selecting currency iSelling price is calculated based on exchange rate of selected currency in selling price from calculated cost price.

Profit amount fixed or percentage applied on selling price only.

Reports available for Booking Reference, Booking Reference With Room List,Customer Group Cost,Customer Quote,Invoice Report and RFQ Quote.

Installed Features

Configurations

User can create contract for Visa, Tickets, Meals, Guides, Transportation, Accommodation for relevant supplier with specific interval.

Contract can't be overlapped for same contract type, interval and Supplier.

Contract must be in Running state for considering prices for Visa, Tickets, Meals, Guides, Transportation, Accommodation.

Contract Creation

Contract must be in Running state for considering prices for defined hotels here.

Add item to specify seasons for selected hotel.

Season Configuration

One have to configure Rate card for season to apply rates for hotel for particular season

As per shown in screen shot

  • Season : defined High,peak,low.
  • Room : Configurable Single,double,Tpl.
  • Room Type : Configurable City view,Pool view,Sea view,suit,superior.
  • Season Interval : Start date and End date.
  • Rates : Rates variance based on combination of Season,Room,Room Type.

Package Creation

For Creation Package Go to Tour & Travels > Package > Package

  • Is Package : If creating package Is package Boolean must have to be True to considered this as a package and if want to display this package on website have to enabled button Published on website.
  • Number of Participant: group of package created for entered number of members specified in this field.
  • Arrival & Departure Date : Generated Itinerary lines automatically from this specified dates, One have to fill other details like description, time and meals.

Hotels

Book hotels for convenient days from Itinerary

  • Category : Select hotel category.
  • Hotel : Hotel filtered based on selected category.
  • Supplier : Select supplier with already contract available in running state between arrival and departure date of package.
  • Days : Select convenient days from Itinerary and other details like room, room type, price will be populated automatically based on selected days from available contract for hotel and its season, One can change this detail also.
  • Generate RFQ : Once package converted to quotation and its confirmed, using this button create Purchase order for specified hotel.

Transportation

Same as Hotel select supplier for available contract in running state between date range of Package. Price and hours auto populated based on selection of product and supplier.

Tickets

Select supplier for available contract in running state between date range of Package. Price auto populated based on selection of product and supplier.

Tours

Select supplier for available contract in running state between date range of Package. Price auto populated based on selection of product and supplier.

Meal

Select supplier for available contract in running state between date range of Package. Price and Days Valid, Menu Item auto populated based on selection of product and supplier.

Guide

Select supplier for available contract in running state between date range of Package. Rate populated based on selection of product and supplier.

Ticketing

Select supplier for available contract in running state between date range of Package. Rate populated based on selection of product and supplier.

Visa

Select supplier for available contract in running state between date range of Package. Price auto populated based on selection of product and supplier.

Passenger

Passenger list auto created when selecting room in hotel. Sequence generation based on room selection SGL,DBL,TPL for number of person going to stay. One have to fill other details passenger name, age, Gender , Nationality and Remarks.

Group Costing

Group Costing:

  • Alternative No of Pax:Number of participant included in specific package.
  • No Of FOC:No of Free of Charge Person of agency going along with Alternative No of Pax of package.
  • Seat Capacity: Transportation Required with specified seat capacity.
  • Group Cost: All cost calculated for No of pax inclusive No Of FOC and Seat capacity.

Cost Price : Cost price calculations based on copmany currency or currency selected in customer's pricelist. All calculation inclusive for hotel, transportation, meals, guide, ticket, ticketing, visa. Cost price categorised for Per Person in DBL, Per Person in single , Per Person in TPL, CWNB.

  • Per Person in DBL:Cost inclusive for hotel room price of double room, Air ticket,tour price, visa price, transportation, guide total, ticket total.
  • Per Person in Single:Cost inclusive for hotel room price of single room, Air ticket,tour price, visa price, transportation, guide total, ticket total.
  • Per Person in TPL:Cost inclusive for hotel room price of TPL room, Air ticket,Tour price, Visa price, Transportation, Guide total, Ticket total.
  • CWNB:Cost inclusive for Meal total ,Guide total , Airicket price , Tour price , Ticket total , Transportation, Visa price.

Selling Price

Selling price is calculated based on exchange rate of selected currency in selling price from calculated cost price.

Profit Price

  • Fixed : Fixed amount applied on Selling price only.
  • Percentage : Entered Percentage calculated for Selling price only.

Note* Except hotel price all prices total divided by number of participantes e.g guide total /number of participantes in package.
